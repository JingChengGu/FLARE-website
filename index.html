Fault Logging and Assessment for Responsive EV Repairs

The FLARE project aims to automate fault detection in EV charging stations using Semantic Segmentation and CNN-based Image Classification. By leveraging real-world data, we help identify charger malfunctions faster and improve maintenance response times.

Model Repo See Demo

Problem

Broken EV chargers (both external and internal issues) make charging unreliable for EV owners and slows down EV adoption.

Solution

AI-driven fault detection using semantic segmentation and binary classifier to speed up the repair process.

Goal

Increases EV charger reliability, strengthens charging infrastructure, and accelerates EV adoption.

Data Collection & Labeling

Data Collection & Labeling

The dataset for training the fault classification model consists of images of EV chargers collected from both manual and online sources, ensuring diverse conditions to improve model generalization. Manual collection involved capturing chargers in various operational states—healthy, broken screen, damaged cord or damaged plug—from different angles, with a focus on front-facing perspectives to better simulate future user photos. Charger locations were identified using PlugShare, prioritizing stations reported as faulty or undergoing maintenance to maximize the presence of malfunctioning units. To further expand the dataset, additional images were sourced online from Google Photos, Reddit, EV forums, and blog posts where users discussed charger failures. These sources provided real-world failure cases, increasing dataset diversity. All online images were manually reviewed to ensure relevance and consistency with manually collected samples.

Methodology

Methodology

We first utilizes NVIDIA’s MiT-B0, a transformer for semantic segmentation. The training data is prepared using Segments.ai, a web-based tool that enables precise pixel-level annotation. Each pixel is classified into one of five categories — Background, Body, Plug, Screen, and Cables. The model outputs a pixel-wise segmentation mask, which helps isolate key charger components before the fault detection stage.
In the fault detection stage, we train a CNN-based image classification model on the isolated components to predict whether they are faulty. Once trained, the classifier integrates with the segmentation model, enabling real-time fault detection and automated flagging for maintenance.
